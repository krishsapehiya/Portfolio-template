Krish — Frontend Developer Portfolio
A sleek, dark-themed personal portfolio website designed for frontend developers and UI/UX designers. Built with clean HTML, CSS, and vanilla JavaScript, it features a fully responsive layout with smooth hover animations and light/dark mode support.
Sections included:

Navbar — Logo, navigation links, theme toggle, and a contact CTA button
Hero — Bold introduction with a call-to-action
About Me — Profile image with spinning badge, skill cards, and tools showcase
Services — Highlighted service cards (Web Design, Mobile App, UI/UX, Graphics)
My Work — Portfolio grid with image cards, overlay info bars, and a "Show more" button
Contact — Contact form or link section

Highlights:

Dark purple color scheme with vibrant pink/violet accents
Serif display typography for headings, clean sans-serif for body
Card hover effects with smooth transitions
Active card highlighting with glowing purple borders
Responsive across desktop, tablet, and mobile
CSS variable-based theming for effortless light/dark switching

Best suited for: Frontend developers, UI/UX designers, and creatives looking for a professional, modern portfolio to showcase their work and skills.
